Developed a Web application using Google App Script for Graduate Advisor to automate the appointment booking, articulation request and study plan submissions. 

Used Google spreadsheets as database and HTML, CSS, JavaScript Cloud to implement the application.

----------------------------------------------------------------------------------------------------------

The project is easily accessible on Google Drive with the Gmail Account:

On logging into the gmail account---> Copy the project to Google Drive

The steps to run the application are:

1. Open the project in the Google Apps Script Dashboard

2. Go to Publish and click ----> Deploy as a web app..

3. Copy and Paste the Current web app URL to open the web app or click on the latest code to directly open the application.
